                                                                    ==================================================
                                                                            Dokumentasi API Unit Test
                                                                    ==================================================

API Unit Test ini menyediakan endpoint-endpoint untuk menguji model analisis sentimen teks dan klasifikasi gambar. Berikut adalah deskripsi lengkap masing-masing endpoint beserta langkah-langkah penggunaannya:

1. Pengujian Analisis Sentimen Teks:
   - Endpoint: /test_sentiment
   - Metode HTTP: POST
   - Deskripsi:
     Endpoint ini digunakan untuk menguji model analisis sentimen teks. Pengguna dapat mengirimkan teks yang ingin dianalisis sentimennya melalui permintaan POST.
   - Request Body:
     {
       "text": "Ini adalah contoh teks yang ingin dianalisis sentimennya."
     }
   - Response:
     - Jika permintaan berhasil, API akan memberikan respons dengan status code 200 dan body JSON berisi hasil analisis sentimen, seperti:
       {
         "sentiment": "positive"
       }
     - Jika terjadi kesalahan dalam permintaan, API akan memberikan respons dengan status code 400 dan body JSON berisi pesan kesalahan yang sesuai.

2. Pengujian Klasifikasi Gambar:
   - Endpoint: /test_image_classification
   - Metode HTTP: POST
   - Deskripsi:
     Endpoint ini digunakan untuk menguji model klasifikasi gambar. Pengguna dapat mengunggah file gambar yang ingin diklasifikasikan melalui permintaan POST.
   - Request Body:
     - Harus berupa file gambar dalam format yang didukung (misalnya, JPEG, PNG, dll.).
   - Response:
     - Jika permintaan berhasil, API akan memberikan respons dengan status code 200 dan body JSON berisi hasil klasifikasi gambar, seperti:
       {
         "predicted_class": "Impressionism",
         "probability": 0.85
       }
     - Jika terjadi kesalahan dalam permintaan, API akan memberikan respons dengan status code 400 dan body JSON berisi pesan kesalahan yang sesuai.

Catatan Penting:
- Pastikan untuk menggunakan metode HTTP yang benar (POST) dan format permintaan yang benar saat melakukan pengujian.
- Pastikan file gambar yang diunggah memiliki format yang didukung untuk memastikan hasil klasifikasi yang akurat.
- API Unit Test ini hanya untuk tujuan pengujian dan evaluasi model. Untuk penggunaan produksi, gunakan endpoint yang sesuai dengan aplikasi utama.
